# Explainable-AI-lab-2

# Лабораторная работа №2: Вскрытие «Черного ящика» (Model-Agnostic методы)

**Выполнила:** Светлана Дрягалова

## Описание работы

Цель работы — изучение методов интерпретации моделей машинного обучения, не зависящих от типа модели (model-agnostic).

В лабораторной работе решаются две бизнес-задачи:

### Задание 1: Иллюзия недвижимости (PDP vs ALE)
Предсказание стоимости квартиры на основе признаков:
- `Area_sqm` — площадь (кв.м)
- `Num_Rooms` — количество комнат
- `Distance_to_metro` — расстояние до метро
- `Floor` — этаж

**Задача:** сравнить Partial Dependence Plot (PDP) и Accumulated Local Effects (ALE) для признака `Area_sqm` и объяснить бизнес-заказчику, какой график корректнее.

### Задание 2: Медицинский консилиум (SHAP и UMAP)
Предсказание риска сердечно-сосудистого заболевания на основе признаков:
- `Age` — возраст
- `Cholesterol` — уровень холестерина
- `Weight` — вес
- `Stress_Level` — уровень стресса

**Задача:**
- Построить глобальную интерпретацию модели с помощью SHAP
- Объяснить предсказания для двух пациентов (высокий и низкий риск)
- Визуализировать пространство признаков с помощью UMAP


## Результаты

### Задание 1
- PDP показывает ложный резкий скачок цены при площади 80–87 кв.м из-за корреляции площади с количеством комнат.
- ALE корректно учитывает эту взаимосвязь и показывает более реалистичную картину.

### Задание 2
- **Глобально:** возраст и уровень стресса — ключевые факторы риска.
- **Пациент с высоким риском (индекс 122):** пожилой возраст (74), высокий холестерин (6.35), высокий стресс (9).
- **Пациент с низким риском (индекс 142):** молодой возраст (25), низкий стресс (1), нормальный вес и холестерин.
- UMAP показал чёткое разделение на два кластера (низкий риск и высокий/пограничный риск).
